# Simple Auth & Portfolio App

This is a simple frontend web application that allows users to **register, log in, and view a portfolio page** after authentication.

The project demonstrates basic usage of **HTML, CSS, JavaScript**, and **database-backed authentication** using **Supabase** as a backend-as-a-service.

- Logged-in user email displayed on portfolio page

- Route protection (portfolio page not accessible without login)
- Responsive and clean UI

- **Authentication & Database:** Supabase  
- **Deployment:** Vercel / Netlify (frontend-only)

---

## ðŸ”¹ How It Works

1. User registers using email and password
2. Credentials are securely handled by Supabase Authentication
3. On successful login, user is redirected to the portfolio page
4. User email is displayed using stored session data
5. Logout clears session and redirects to login page
